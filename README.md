# 🚀 Depude – Duplicate Data Remover

Depude is a simple and efficient web application that helps users remove duplicate records from CSV files. Built using Flask and Pandas, it provides a fast and user-friendly interface for data cleaning.

---

## 📌 Features

- 📂 Upload CSV files
- 🧹 Automatically removes duplicate records
- ⚡ Fast processing using Pandas
- 📥 Download cleaned file instantly
- 🎨 Clean and modern UI

---

## 🛠️ Tech Stack

- Python
- Flask
- Pandas
- HTML/CSS
